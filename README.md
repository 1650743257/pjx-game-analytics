# pjx-game-analytics

    本项目对一款IAA手游的用户行为数据进行深度分析（辨识度较高的数据和内容均已脱敏）。游戏包含多个场景、游戏回合、广告变现模块以及丰富的用户交互功能。通过数据分析，旨在理解用户行为模式、评估广告效果、优化游戏体验，为产品迭代提供数据支持，并根据分析结果提供了一个用于优化的A/B测试方案。


技术栈：
1. Python：数据清洗、数据格式化、数据表拆分
2. SQLite：数据库管理、简单查询
3. DuckDB：复杂查询
4. Excel：分析结果可视化


附文档：
1. SQL代码_脱敏.sql  
数据清洗：多表合并去重、无效ID过滤、空值处理、数据质量评估。  
用户分析：日活/周活/月活、新增用户、流失用户、留存率计算、用户等级分层。  
行为分析：会话时长分布、事件间隔、游戏回合时长与胜率、投入值分布。  
广告效果：全屏广告完播率、激励视频触发与完播分析。  
按钮交互：各按钮点击次数与用户等级、广告行为的关系分析。

2. XX项目埋点数据分析报告_脱敏.pdf  
报告基于埋点数据，分析了用户会话、流失因素、广告变现及玩家行为。核心发现包括：退出主因是局内失败与广告反感；连败与高风险投注显著提升流失风险；激励视频在特定场景转化率高。报告提出优化败局分布、强化连败补偿、动态调控广告、增强短时正反馈等策略，以提升留存与变现效率。

3. merge_deduplicate.py  
合并原始数据并去重。

4. clean.py  
剔除无效id对应事件。

5. split.py  
解析数据中的参数列为多列，并按事件拆分为多个子表。

6. 保险机制AB测试方案.pdf  
针对分析中提出的优化机制，设计A/B测试方案，期望降低玩家流失。包括背景与目标、实验设计、机制描述、埋点设计、指标体系、分析方法、实验周期、预期结论，完整覆盖从假设到验证的全流程。
